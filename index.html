// Sensibilidade ao comprimento L — 3 métodos
● Darcy-Weisbach ● Azevedo Netto ● Torricelli (Cd)
Torricelli é linha horizontal — não depende de L. Azevedo Netto cai lentamente (∝ 1/√L). Darcy-Weisbach cai mais rápido pois modela o atrito real: quanto mais longo, mais energia dissipada.
// Sensibilidade ao diâmetro D — 3 métodos
● Darcy-Weisbach ● Azevedo Netto ● Torricelli (Cd)
Todos crescem com D, mas Torricelli só ganha área (∝ D²). D-W e A-N ganham área E menor atrito, crescendo mais rápido. Os 3 convergem quando L→0.
